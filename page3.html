Position Property

relative: Positions the element relative to its normal position. It shifts based on offsets (top, left, etc.) without affecting other elements.

absolute: Positions the element relative to the nearest positioned ancestor. If none exists, it’s relative to the document body. It ignores normal document flow.

fixed: Pins the element to a specific spot on the viewport. It stays in place even when scrolling the page.

sticky: The element toggles between relative and fixed based on the scroll position. It “sticks” to a defined position within its container when scrolling.

Display Property

inline Elements flow horizontally without breaking lines. You can’t set width or height. Example: span.

block: Elements occupy the full width of their container by default and allow setting width and height. Example: div.

inline-block: Similar to inline, but allows setting width and height like block elements while staying in line with other elements.

Float & Clear Example

The float property shifts an element to the left or right, allowing text and inline elements to wrap around it. It was initially used for text wrapping but is now less common with the rise of Flexbox and Grid.

The clear property prevents elements from wrapping around floated elements. It ensures that an element stays below (or beside) floated content. Values include left, right, both, or none.
